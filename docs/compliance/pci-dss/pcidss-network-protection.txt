Requirement 1: Install and Maintain Network Security Controls.

1.2 Network security controls (NSCs) are configured and maintained.
1.2.1 Configuration standards for NSCs are defined, implemented, and maintained.
1.2.2 All changes to network connections and to configurations of NSCs are approved and managed.

1.3 Network access to and from the cardholder data environment (CDE) is restricted.
1.3.1 Inbound traffic to the CDE is restricted to only traffic that is necessary.
1.3.2 Outbound traffic from the CDE is restricted to only traffic that is necessary.

1.4 Network connections between trusted and untrusted networks are controlled.
1.4.1 NSCs are implemented between trusted and untrusted networks.
1.4.2 Inbound internet traffic is limited to IP addresses within the DMZ.

Requirement 2: Apply Secure Configurations to All System Components.

2.2 System components are configured and managed securely.
2.2.2 Vendor default accounts and passwords are changed or removed before installing a system on the network.
2.2.3 Primary functions are isolated to prevent components with different security levels from co-existing on the same system.
2.2.4 System components are configured to enforce only necessary services, protocols, and daemons.
